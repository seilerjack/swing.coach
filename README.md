# swing.coach
SwingCoach Web is a browser-based golf swing analysis platform that uses video and AI-driven metrics to provide personalized, actionable feedback. Users upload a swing, report shot outcomes, and receive visual overlays, key metrics, and supportive coaching tips to improve their game.

## Overview
SwingCoach Web is a full-stack web application that analyzes golf swing videos, extracts biomechanical metrics, and generates personalized coaching feedback using AI. Users upload swing videos, enter basic shot information, and receive detailed swing analysis, mechanical insights, and actionable improvement tips. The app combines computer vision, motion-analysis algorithms, and GPT-powered coaching to deliver an interactive and beginner-friendly digital swing coach.
